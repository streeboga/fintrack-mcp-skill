# FinTrack MCP Server

MCP сервер для управления личными финансами в приложении FinTrack.

## Когда Применять

Используй этот skill когда:
- Нужно создать, получить, изменить или удалить финансовые данные
- Работаешь с MCP инструментами fintrack
- Создаешь новые MCP Tools для FinTrack
- Отлаживаешь проблемы с MCP сервером

## Основные Сущности

| Сущность | Описание | Tools |
|----------|----------|-------|
| Accounts | Банковские счета, карты, наличные | 6 tools |
| Categories | Категории доходов/расходов | 6 tools |
| Transactions | Финансовые операции | 6 tools |
| Recurrings | Повторяющиеся платежи | 6 tools |
| Tags | Метки для организации | 6 tools |
| Projects | Проекты и подпроекты | 6 tools |

**Итого: 36 инструментов**

## Паттерн Создания

**ВСЕГДА** используй двухшаговый процесс для создания:
1. `prepare-{entity}` → валидация и preview
2. Подтверждение пользователя
3. `create-{entity}` → создание

## Паттерн Обновления

Для изменения — передавай только изменяемые поля:
1. `update-{entity}` с `{entity}_id` + нужные поля

## Паттерн Удаления

Для удаления — подтверди с пользователем, затем:
1. `delete-{entity}` с `{entity}_id`

## Быстрая Справка

### Суммы
- В копейках/центах: 10000 = 100.00 RUB

### Даты
- ISO 8601: YYYY-MM-DD

### Типы транзакций
- `expense` - расход
- `income` - доход
- `transfer` - перевод (только recurring)

### Типы счетов
- `checking`, `savings`, `credit`, `cash`, `investment`, `loan`, `other`

### Группы счетов
- `cash`, `credit`, `savings`, `investment`, `loan`, `other`

### Типы категорий
- `expense`, `income`, `both`
